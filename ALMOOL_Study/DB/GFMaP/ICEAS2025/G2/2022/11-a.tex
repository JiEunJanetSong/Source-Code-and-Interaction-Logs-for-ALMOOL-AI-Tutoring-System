(a) Given that $\cos \left(2 x+15^{\circ}\right)=\sin \left(x-30^{\circ}\right), 0^{\circ} \leq x \leq 90^{\circ}$, find the value of $x$.

}\SetValue{Translation}{%

}\SetValue{Hint}{%



}\SetValue{GPTSolution}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

We want to solve the equation

$
\cos\!\bigl(2x + 15^\circ\bigr) \;=\; \sin\!\bigl(x - 30^\circ\bigr),
\quad 0^\circ \le x \le 90^\circ.
$

1. Convert the sine function to a cosine function  
   Recall \(\sin \theta = \cos(90^\circ - \theta)\). Hence,
   $
   \sin(x - 30^\circ) \;=\; \cos\bigl(90^\circ - (x - 30^\circ)\bigr)
                           \;=\; \cos(120^\circ - x).
   $
   Therefore, the equation becomes
   $
   \cos\!\bigl(2x + 15^\circ\bigr)
   \;=\; \cos\!\bigl(120^\circ - x\bigr).
   $

2. Solve using \(A+B=90^\circ,\quad then \sin A=\cos B\) 

$2 x+15^{\circ}+x-30^{\circ}=90^{\circ}$,

$
\boxed{x = 35^\circ}.
$

}\SetValue{Answer}{%

35
