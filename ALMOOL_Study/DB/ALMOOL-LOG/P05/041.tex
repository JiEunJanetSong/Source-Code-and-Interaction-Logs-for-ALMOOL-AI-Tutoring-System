\begin{GPTQ}
    시작하기 전에 내가 알고 있어야 하는 공식이 있을까?   
\end{GPTQ}

\begin{GPTA}
물론입니다! 이 문제를 풀기 위해 알아두면 유용한 몇 가지 공식을 소개합니다:

    1. 삼각형의 내각 합의 정리: 모든 삼각형의 내각의 합은 $180^{\circ}$ 입니다.

    2. 사인 값의 활용: 삼각형에서 한 각의 사인 값은 마주 보는 변의 길이를 빗변의 길이로 나눈 값과 같습니다. 이는 직각삼각형의 반대 변과 빗변 간의 관계를 구할 때 사용됩니다.
    
    $$
    \sin \theta=\frac{\text { 마주보는 변의 길이 }}{\text { 빗변의 길이 }}
    $$
    
    3. 역삼각함수: 주어진 사인 값이나 코사인 값을 이용하여 각도를 구할 때 사용하는 함수입니다. 예를 들어, $\theta=\sin ^{-1}($ value $)$ 입니다.
    
    이 공식들을 사용하여 주어진 삼각형 문제를 해결할 수 있습니다. 도움이 필요하시면 언제든지 말씀하세요! 
\end{GPTA}
