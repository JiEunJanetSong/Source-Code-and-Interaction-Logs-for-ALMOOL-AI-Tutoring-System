 순서대로 풀어줘

\begin{GPTA}
    물론입니다. 주어진 문제를 순서대로 풀어보겠습니다.

    문제: $x$ 와 $n$ 이 1 보다 큰 수일 때, $x$ 가 2 에서 100 으로 변할 때 $y$ 값이 가장 많이 증가하는 표현식을 고르세요. 주어진 선택지:

    - (A) $y=n x+3$

    - (B) $y=-n x+3$

    - (C) $y=x^n+3$

    - (D) $y=x^{-n}+3$
    
    풀이: 각 식에 대해 $x=2$ 와 $x=100$ 을 대입해서 비교해보겠습니다. 여기서는 $n$ 이 3 이라고 가정합니다.

    - 식 (A):
    
    $$
    y=n x+3 y \quad=3 \times 2+3=9 \quad(\text { when } x=2, n=3) y=3 \times 100+3=303 \quad(\text { when } x=100, n=3)
    $$
    
    - 식 (B):
    
    $$
    y=-n x+3 y \quad=-3 \times 2+3=-3 \quad(\text { when } x=2, n=3) y=-3 \times 100+3=-297 \quad(\text { when } x=100, n=3)
    $$
    
    - 식 $(\mathrm{C})$ :
    
    $$
    y=x^n+3 y \quad=2^3+3=11 \quad(\text { when } x=2, n=3) y=100^3+3=1,000,003 \quad(\text { when } x=100, n=3)
    $$
    
    - 식 (D):
    
    $$
    y=x^{-n}+3 y \quad=2^{-3}+3=3 \frac{1}{8} \quad(\text { when } x=2, n=3) y=100^{-3}+3=3.000001 \quad(\text { when } x=100, n=3)
    $$
    
    
    결과를 보면 식 (C) $y=x^n+3^{\circ}$ 이 $x=2$ 일 때 11 이고, $x=100$ 일 때 $1,000,003$ 로 가장 큰 증가를 보입니다. 그래서 답은 (C) 입니다.    
\end{GPTA}
