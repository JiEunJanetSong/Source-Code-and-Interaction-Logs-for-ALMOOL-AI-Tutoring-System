물론이죠! 문제를 단계적으로 해결해 보겠습니다:

주어진 방정식은 다음과 같습니다:

1. 공급 방정식:

$$
\text { 가격 }=30-\text { 수량 }
$$

2. 수요 방정식:

$$
\text { 가격 }=10+3 \times \text { 수량 }
$$


목표: 공급과 수요가 균형 상태일 때의 수량을 찾는 것입니다. 이를 위해 두 방정식을 가격에 대해 같게 설정하고 수량을 구합니다.

단계별 풀이:

1. 두 가격 방정식을 같게 설정합니다:

$$
30-\text { 수량 }=10+3 \times \text { 수량 }
$$

2. 방정식을 정리합니다:

$$
30-10=3 \times \text { 수량 }+ \text { 수량 }
$$

3. 왼쪽에서 오른쪽으로 20 을 얻습니다:

$$
20=4 \times \text { 수량 }
$$

4. 양변을 4로 나누어 수량을 구합니다:

$$
\text { 수량 }=5
$$
