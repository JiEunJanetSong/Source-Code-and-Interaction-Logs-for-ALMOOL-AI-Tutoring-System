\begin{GPTQ}
    각이 똑같으면 사인값도 같아?     
\end{GPTQ}

\begin{GPTA}
    네, 맞습니다! 두 각이 동일한 경우, 즉 그 각듈의 크기가 같다면 사인 값도 같습니다.

    이를 이해하기 위해 $\sin (\theta)$ 의 정의를 살펴보면, 주어진 각 $\theta$ 에 대해서 직각삼각형에서 사인은 $\frac{\text { opposite }}{\text { hypotenuse }}$ 의 비율로 정의 됩니다. 각의 크기가 일정하면 해당 비율도 항상 일정 한 값을 가집니다. 따라서 각이 동일할 경우, 그 각의 사인 값도 동일합니다.

    즉, $\theta_1=\theta_2$ 이면 $\sin \left(\theta_1\right)=\sin \left(\theta_2\right)$ 가 됩니다. 닮은 삼각형에서는 대응하는 각들이 같으므로, 대응 하는 각들의 사인 값도 같습니다.
    \end{GPTA}
