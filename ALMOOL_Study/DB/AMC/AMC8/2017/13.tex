Peter, Emma, and Kyler played chess with each other. Peter won 4 games and lost 2 games. Emma won 3 games and lost 3 games. If Kyler lost 3 games, how many games did he win?

\begin{center}		
    \includegraphics[scale=0.6]{AMC-8-pics/2017-13.png}
	\end{center}

\ansFIVEs{%start
		0 }{%<--- (A)
		1 }{%<--- (B)
		2 }{%<--- (C)
		3 }{%<---- (D)
		4 }%<---- (D)
}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Answer (B): Whenever a person wins a game, another person loses that game. So the total number of wins equals the total number of losses. Peter, Emma, and Kyler lost $2+3+3=8$ games altogether, and Peter and Emma won $4+3=7$ games in total. Therefore, Kyler won 1 game.
