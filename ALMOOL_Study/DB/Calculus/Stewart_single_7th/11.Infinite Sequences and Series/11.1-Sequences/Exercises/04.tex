List the first five terms of the sequence.

$$a_n=\frac{3^n}{1+2^n}$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Let's calculate the first five terms of the sequence $a_n=\frac{3^n}{1+2^n}$ by substituting n = 1, 2, 3, 4, and 5.

1.  For n = 1:

    $$a_1 = \frac{3^1}{1+2^1} = \frac{3}{1+2} = \frac{3}{3} = 1$$

2.  For n = 2:

    $$a_2 = \frac{3^2}{1+2^2} = \frac{9}{1+4} = \frac{9}{5}$$

3.  For n = 3:

    $$a_3 = \frac{3^3}{1+2^3} = \frac{27}{1+8} = \frac{27}{9} = 3$$

4.  For n = 4:

    $$a_4 = \frac{3^4}{1+2^4} = \frac{81}{1+16} = \frac{81}{17}$$

5.  For n = 5:

    $$a_5 = \frac{3^5}{1+2^5} = \frac{243}{1+32} = \frac{243}{33} = \frac{81}{11}$$ (simplified by dividing numerator and denominator by 3)

The first five terms of the sequence are:

$$1, \frac{9}{5}, 3, \frac{81}{17}, \frac{81}{11}$$
