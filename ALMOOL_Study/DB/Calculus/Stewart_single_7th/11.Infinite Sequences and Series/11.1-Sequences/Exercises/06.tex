List the first five terms of the sequence.

$$a_n=\cos \frac{n \pi}{2}$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Let's find the first five terms of the sequence defined by the formula $a_n=\cos \frac{n \pi}{2}$. We will substitute n = 1, 2, 3, 4, and 5 into the formula.

1.  For n = 1:

    $$a_1 = \cos\left(\frac{1 \cdot \pi}{2}\right) = \cos\left(\frac{\pi}{2}\right) = 0$$

2.  For n = 2:

    $$a_2 = \cos\left(\frac{2 \cdot \pi}{2}\right) = \cos(\pi) = -1$$

3.  For n = 3:

    $$a_3 = \cos\left(\frac{3 \cdot \pi}{2}\right) = \cos\left(\frac{3\pi}{2}\right) = 0$$

4.  For n = 4:

    $$a_4 = \cos\left(\frac{4 \cdot \pi}{2}\right) = \cos(2\pi) = 1$$

5.  For n = 5:

    $$a_5 = \cos\left(\frac{5 \cdot \pi}{2}\right)$$

    Since the cosine function has a period of $2\pi$, we can write $\frac{5\pi}{2} = 2\pi + \frac{\pi}{2}$.

    Therefore, $a_5 = \cos\left(2\pi + \frac{\pi}{2}\right) = \cos\left(\frac{\pi}{2}\right) = 0$.

The first five terms of the sequence are:

$$0, -1, 0, 1, 0$$
