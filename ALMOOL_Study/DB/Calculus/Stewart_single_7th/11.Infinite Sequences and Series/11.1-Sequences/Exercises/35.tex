Determine whether the sequence converges or diverges. If it converges, find the limit.

$$a_n=\cos \left(\frac{n}{2} \right)$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

We are given the sequence:

$
a_n = \cos\left(\frac{n}{2}\right)
$

Step 1: Understand the Behavior

The cosine function is periodic with period $2\pi$, and the input to the cosine here is:

$
\frac{n}{2}, \text{ where } n = 1, 2, 3, \dots
$

So the inputs to $\cos(x)$ are:

$
\frac{1}{2},\ \frac{2}{2} = 1,\ \frac{3}{2},\ 2,\ \frac{5}{2},\ \dots
$

These values do not approach a finite limit, and are not rational multiples of $\pi$, so the outputs of $\cos\left(\frac{n}{2}\right)$ will oscillate in an irregular, non-repeating way.

Step 2: Use Bounded Oscillation

- Since cosine values always lie in the interval $[-1, 1]$, the sequence is bounded.

- But since the values of $\frac{n}{2}$ get arbitrarily far from each other and are not approaching any single value modulo $2\pi$, the sequence $\cos\left(\frac{n}{2}\right)$ does not settle.

This type of sequence is dense in $[-1, 1]$, and continues to fluctuate.

Final Answer:

- The sequence does not converge

- It diverges due to oscillation

$
\boxed{\text{The sequence diverges}}
$
