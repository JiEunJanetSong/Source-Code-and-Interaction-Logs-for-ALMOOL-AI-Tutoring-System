List the first five terms of the sequence.

$$
a_1=2, \quad a_{n+1}=\frac{a_n}{1+a_n}
$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%


This sequence is defined recursively. We start with $a_1$ and use the formula $a_{n+1}=\frac{a_n}{1+a_n}$ to find the subsequent terms.

1.  a₁: The first term is given directly.

    $$a_1 = 2$$

2.  a₂: Use the recursive formula with $n=1$.

    $$a_2 = a_{1+1} = \frac{a_1}{1+a_1} = \frac{2}{1+2} = \frac{2}{3}$$

3.  a₃: Use the recursive formula with $n=2$.

    $$a_3 = a_{2+1} = \frac{a_2}{1+a_2} = \frac{2/3}{1+2/3}$$

    To simplify, find a common denominator for the bottom part:
    
    $1 + 2/3 = 3/3 + 2/3 = 5/3$.

    $$a_3 = \frac{2/3}{5/3} = \frac{2}{3} \times \frac{3}{5} = \frac{2}{5}$$

4.  a₄: Use the recursive formula with $n=3$.

    $$a_4 = a_{3+1} = \frac{a_3}{1+a_3} = \frac{2/5}{1+2/5}$$

    Simplify the denominator: $1 + 2/5 = 5/5 + 2/5 = 7/5$.

    $$a_4 = \frac{2/5}{7/5} = \frac{2}{5} \times \frac{5}{7} = \frac{2}{7}$$

5.  a₅: Use the recursive formula with $n=4$.

    $$a_5 = a_{4+1} = \frac{a_4}{1+a_4} = \frac{2/7}{1+2/7}$$

    Simplify the denominator: $1 + 2/7 = 7/7 + 2/7 = 9/7$.

    $$a_5 = \frac{2/7}{9/7} = \frac{2}{7} \times \frac{7}{9} = \frac{2}{9}$$

The first five terms of the sequence are:

$$2, \frac{2}{3}, \frac{2}{5}, \frac{2}{7}, \frac{2}{9}$$
