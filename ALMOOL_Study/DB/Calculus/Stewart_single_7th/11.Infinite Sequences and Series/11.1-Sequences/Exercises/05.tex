List the first five terms of the sequence.

$$a_n=\frac{(-1)^{n-1}}{5^n}$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Let's find the first five terms of the sequence defined by the formula $a_n=\frac{(-1)^{n-1}}{5^n}$. We will substitute n = 1, 2, 3, 4, and 5.

1.  For n = 1:

    $$a_1 = \frac{(-1)^{1-1}}{5^1} = \frac{(-1)^0}{5} = \frac{1}{5}$$

    (Remember that any non-zero number raised to the power of 0 is 1)

2.  For n = 2:

    $$a_2 = \frac{(-1)^{2-1}}{5^2} = \frac{(-1)^1}{25} = -\frac{1}{25}$$

3.  For n = 3:

    $$a_3 = \frac{(-1)^{3-1}}{5^3} = \frac{(-1)^2}{125} = \frac{1}{125}$$

4.  For n = 4:

    $$a_4 = \frac{(-1)^{4-1}}{5^4} = \frac{(-1)^3}{625} = -\frac{1}{625}$$

5.  For n = 5:

    $$a_5 = \frac{(-1)^{5-1}}{5^5} = \frac{(-1)^4}{3125} = \frac{1}{3125}$$

The first five terms of the sequence are:

$$\frac{1}{5}, -\frac{1}{25}, \frac{1}{125}, -\frac{1}{625}, \frac{1}{3125}$$
