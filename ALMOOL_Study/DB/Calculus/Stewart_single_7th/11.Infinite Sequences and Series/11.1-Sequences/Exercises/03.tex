List the first five terms of the sequence.

$$a_n=\frac{2n}{n^2+1}$$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Let's find the first five terms of the sequence defined by the formula $a_n=\frac{2n}{n^2+1}$. We do this by substituting the values n = 1, 2, 3, 4, and 5 into the formula.

1.  For n = 1:

    $$a_1 = \frac{2(1)}{1^2+1} = \frac{2}{1+1} = \frac{2}{2} = 1$$

2.  For n = 2:

    $$a_2 = \frac{2(2)}{2^2+1} = \frac{4}{4+1} = \frac{4}{5}$$

3.  For n = 3:

    $$a_3 = \frac{2(3)}{3^2+1} = \frac{6}{9+1} = \frac{6}{10} = \frac{3}{5}$$ (simplified)

4.  For n = 4:

    $$a_4 = \frac{2(4)}{4^2+1} = \frac{8}{16+1} = \frac{8}{17}$$

5.  For n = 5:

    $$a_5 = \frac{2(5)}{5^2+1} = \frac{10}{25+1} = \frac{10}{26} = \frac{5}{13}$$ (simplified)

The first five terms of the sequence are:

$$1, \frac{4}{5}, \frac{3}{5}, \frac{8}{17}, \frac{5}{13}$$
