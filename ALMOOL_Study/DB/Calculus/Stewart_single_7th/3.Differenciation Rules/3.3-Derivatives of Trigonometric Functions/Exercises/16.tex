Differentiate.

$y=x^2 \sin x \tan x$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Using the Product Rule for three factors:

Let
$
y = x^2 \sin x \,\tan x.
$

Think of $y$ as the product $f(x)\,g(x)\,h(x)$, where
$
f(x) = x^2, \quad g(x) = \sin x, \quad h(x) = \tan x.
$

The derivative of a product of three factors is
$
\dfrac{d}{dx}\bigl(fg h\bigr) 
= f'gh + fg'h + fgh'.
$

Compute each derivative:

1. $f'(x) = \dfrac{d}{dx}(x^2) = 2x.$  

2. $g'(x) = \dfrac{d}{dx}(\sin x) = \cos x.$ 
 
3. $h'(x) = \dfrac{d}{dx}(\tan x) = \sec^2 x.$

Hence,
$
y' 
= (2x)\,\sin x \,\tan x \;+\; x^2\,(\cos x)\,\tan x \;+\; x^2\,\sin x\,(\sec^2 x).
$

That is:
$
\boxed{
y' = 2x \,\sin x \,\tan x \;+\; x^2 \,\cos x \,\tan x \;+\; x^2 \,\sin x \,\sec^2 x.
}
$
