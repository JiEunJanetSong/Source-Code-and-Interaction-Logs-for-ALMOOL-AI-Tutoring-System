Differentiate.

$y=2 \sec x-\csc x$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

Given
$
y = 2\sec x - \csc x,
$
take the derivative term by term:

1. $\dfrac{d}{dx}(2\sec x) = 2\sec x \tan x.$  

2. $\dfrac{d}{dx}(-\csc x) = -(-\csc x \cot x) = \csc x \cot x.$

Hence,
$
y' = 2\sec x \tan x + \csc x \cot x.
$
