Find the limit.

$\lim _{x \rightarrow 0} \dfrac{\sin \left(x^2\right)}{x}$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

We want 
$
\lim_{x \to 0} \dfrac{\sin(x^2)}{x}.
$

For small $x$, note that $\sin(x^2) \approx x^2$. Thus,

$
\dfrac{\sin(x^2)}{x} \approx \dfrac{x^2}{x} = x,
$
which goes to $0$ as $x \to 0$. 

More rigorously, we can use the standard limit result $\lim_{u \to 0} \dfrac{\sin u}{u} = 1$ by setting $u = x^2$. 

Then

$
\dfrac{\sin(x^2)}{x} 
= \dfrac{\sin(x^2)}{x^2} \cdot x 
= \left( \dfrac{\sin(x^2)}{x^2} \right) x.
$

As $x \to 0$, $x^2 \to 0$, so $\dfrac{\sin(x^2)}{x^2} \to 1$. 

Hence the expression behaves like $x \cdot 1$, which goes to $0$.

Therefore,
$
\boxed{0}.
$
