Find the derivative of the function.

$f(x)=\dfrac{1}{(1+\sec x)^2}$

}\SetValue{Concept}{%



}\SetValue{AltText}{%



}\SetValue{Solution}{%

We can rewrite the function as 

$$
f(x) = \bigl(1 + \sec x\bigr)^{-2}.
$$

Define

$$
u = 1 + \sec x \quad \Longrightarrow \quad f(x) = u^{-2}.
$$

Then,

$$
\dfrac{du}{dx} = \sec x \tan x.
$$

Using the chain rule,

$$
f'(x) 
= -2 \, (1 + \sec x)^{-3} \cdot \dfrac{d}{dx}[1 + \sec x] 
= -2 \, (1 + \sec x)^{-3} \bigl(\sec x \tan x\bigr).
$$

Hence,

$$
\boxed{f'(x) = -2\,\dfrac{\sec x \tan x}{\bigl(1 + \sec x\bigr)^3}.
}
$$
