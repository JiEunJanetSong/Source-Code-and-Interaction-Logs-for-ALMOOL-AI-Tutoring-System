What are the possible values of $b$ in the system of equations below?

\begin{align*}
    &b = 2a + 1 \\
    &b + 2 = a^2
    \end{align*}
    
 
\ansFOURs{%start
-3,-7}{%<--- (A)
-1,7}{%<--- (B)
3,-1}{%<--- (C)
3,7}%<---- (D)

}\SetValue{Solution}{%

Answer: \textbf{(B)} 

We are given the system of equations:

$
b = 2a + 1
$

$
b + 2 = a^2
$

 Step 1: Express $b$ in Terms of $a$
From the first equation:

$
b = 2a + 1
$

Substituting this into the second equation:

$
(2a + 1) + 2 = a^2
$

$
2a + 3 = a^2
$

 Step 2: Rearrange into Quadratic Form
Rearrange the equation:

$
a^2 - 2a - 3 = 0
$

 Step 3: Solve for $a$
Factorize:

$
(a - 3)(a + 1) = 0
$

Thus, the possible values of $a$ are:

$
a = 3 \quad \text{or} \quad a = -1
$

 Step 4: Find Corresponding Values of $b$
Using $b = 2a + 1$:

\begin{itemize}
    \item If $a = 3$:

    $$
    b = 2(3) + 1 = 6 + 1 = 7
    $$
    \item If $a = -1$:

    $$
    b = 2(-1) + 1 = -2 + 1 = -1
    $$
\end{itemize}

 Step 5: Select the Correct Answer
The possible values of $b$ are $-1, 7$, so the correct choice is:

$
\mathbf{B) -1, 7}
$
